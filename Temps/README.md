
# 一些常用的正则表达式
* a-z A-Z _ 0-9  //最常见的字符 
* (abc) (skd)  //用圆括号包含起来的单元符合 
* [abcs] [^abd]  //用方括号包含的原子表，原子表中的^代表排除或相反内容 
* 转义字符 
  * \d  包含所有数字[0-9] 
  * \D  除所有数字外[^0-9] 
  * \w  包含所有英文字符[a-zA-Z_0-9] 
  * \W  除所有英文字符外[^a-zA-Z_0-9]
  * \s   包含空白区域如回车、换行、分页等 [\f\n\r] 



# 重复次数
* +等价于{1,}
* *等价于{0,}
* ?等价于{0,1}
<br>**必须要逗号** 


# 贪婪和非贪婪
* .*贪婪 会匹配到最后一个符合的
* .*?非贪婪 会匹配到最近的一个符合的
